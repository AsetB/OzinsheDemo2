# Ozinshe
Video streaming service for kids on Kazakh language.
UIKit with SnapKit.

<img src="https://github.com/AsetB/OzinsheDemo2/blob/main/ScreenRecording.gif" width="256">

[App Store](https://apps.apple.com/us/app/ozinshe/id1630335072)

Dependencies:
 - Alamofire
 - KeychainSwift
 - Localize_Swift
 - SDWebImage
 - SnapKit
 - SVProgressHUD
 - SwiftyJSON
 - YouTubePlayerKit
