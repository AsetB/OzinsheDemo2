Internship project. UIKit with SnapKit
[App Store](https://apps.apple.com/us/app/ozinshe/id1630335072)
Dependencies:
 - Alamofire
 - KeychainSwift
 - Localize_Swift
 - SDWebImage
 - SnapKit
 - SVProgressHUD
 - SwiftyJSON
 - YouTubePlayerKit
